# Academic Coding

Welcome to the Academic Coding repository!  
This repository contains various assignments, lab work, and projects from SOA ITER, organized into semester-wise folders for easy access.

---

## Semester 5

This semester includes core subjects that strengthen your foundation in programming, operating systems, networking, Python, and machine learning concepts.

### Subjects

1. **Practical Programming with C**  
Covers arrays, strings, file handling, pointers, concurrency, searching and sorting, graphs, data structures, and dynamic memory allocation.  

2. **Design Principles of Operating Systems**  
Focuses on process scheduling, concurrency, synchronization, memory management, virtual memory, I/O management, and file systems.  

3. **Computer Networking: Concepts**  
Explores OSI and TCP/IP models, IPv4/IPv6 addressing, subnetting, routing, wireless standards, cloud computing, and network security.  

4. **Python for Computer Science and Data Science 1**  
Introduces Python programming with data types, control statements, functions, OOPs, and data structures. Practical exercises focus on Python applications for data analysis and problem-solving.  

5. **Machine Learning Concepts 1**  
Introduces ML fundamentals such as data preparation, feature engineering, classification, and regression. Hands-on labs focus on implementing supervised learning methods.  

---

## Semester 6

This semester builds on your previous knowledge and introduces advanced topics and applications in programming, machine learning, Python, databases, and network security.

### Subjects

1. **Game Programming with C**  
Focuses on applying C programming skills to create interactive games and simulations.  

2. **Machine Learning Concepts 2**  
Covers advanced machine learning techniques, including unsupervised learning, model evaluation, and optimization.  

3. **Python for Computer Science and Data Science 2**  
Builds on Python foundations to implement advanced data science projects, including data visualization, analysis, and machine learning pipelines.  

4. **Introduction to Databases**  
Introduces relational databases, SQL queries, normalization, indexing, and basic database design concepts.  

5. **Computer Networking Security**  
Covers network security principles, encryption, firewalls, secure protocols, and practical implementation of network protection mechanisms.  

---

Organizing your work semester-wise allows for easier navigation and management of assignments, labs, and projects.
