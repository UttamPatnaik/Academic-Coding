5. Game loop with ESC key exit
while (window.isOpen())
{
    sf::Event event;

    while (window.pollEvent(event))
    {
        if (event.type == sf::Event::Closed)
            window.close();
    }

    if (sf::Keyboard::isKeyPressed(sf::Keyboard::Escape))
    {
        window.close();
    }
}

9. Time & bit rate calculation

Speed = 200 pixels/sec
Distance = 2000 pixels

Time:

Time = Distance / Speed
     = 2000 / 200
     = 10 seconds

Bit rate:

1 pixel = 8 bits

Bits per second = 200 × 8
                = 1600 bps
10. Distance per frame

Let time per frame = t seconds

Distance = speed × time
         = beeSpeed × t

If using clock:

float dt = clock.restart().asSeconds();
float distance = beeSpeed * dt;
